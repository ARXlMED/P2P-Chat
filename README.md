# P2P-Chat
## What is this application? / Что из себя представляет данное приложение?
Данное приложение это P2P чат, написанный на C# с использованием графического интерфейса WPF. Подключенные копии программы могут хранить историю сообщений для узлов которые подключились к чату после того как сообщения были написаны. Приложение использует UDP Broadcast для поиска узлов в локальной сети уже подключенные к чату, после получившие Broadcast узлы отправляют UDP пакет направленный на адрес с которого пришёл Broadcast со своими данными, а после узел получает обратные UDP и устанавливает TCP соединения с узлами который отправили UDP ответы. Свой IP адрес, имя и UDP с TCP портом задаются при запуске приложения в отдельном окне.
## Install / Установка
1. Установить копии репозитория на все необходимые устройства
2. Запустить приложения и указать на всех локальные IP адреса, находящиеся в общей локальной сети
3. Наслаждаться диалогом с вашими собеседниками
